# PulseGuard ⚡

**PulseGuard** is a lightweight, single-purpose predictive utility concept designed to eliminate unexpected prepaid electricity blackouts. Instead of relying on complex hardware modifications or live utility API access, PulseGuard acts as an intelligent virtual ledger that calculates your household's dynamic consumption rate to warn you precisely when it's time to recharge.

---

## 🚀 Key Features

*   **Dual-Interface Dashboard:** Features an immediate visual "Fuel Gauge" circular dial that dynamically changes color states alongside a prominent "Days Remaining" countdown text display.
*   **Proactive Smart Alerts:** Automatically calculates when your virtual ledger is projected to hit **5.0 units** and triggers a high-priority early warning notification 24 hours in advance.
*   **Dynamic Consumption Tracking:** Calibrates your exact daily burn rate automatically based on usage intervals without requiring manual inputs.
*   **Zero-Bloat Scope:** Intentionally designed around the "Unix philosophy"—doing exactly one critical job flawlessly with zero feature creep.

---

## 🛠️ How to Run the Simulator Locally

1. Clone or download the `index.html` file to your machine.
2. Double-click the file to open it instantly in any modern web browser.
3. Use the **Simulation Panel** to adjust daily drain rates, load mock units, or fast-forward time to test the dynamic alert system.
